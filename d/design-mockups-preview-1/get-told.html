Get-told — a durable, addressed inbox

Today there's no delivery channel — no push, no badge that means anything, no durable inbox. Settings sells three notification engines wired to nothing, and the header bell (confirmed in app.js) surfaces Mr. Wrangler fix-ticket resolutions, not what a worker needs to do next. This mockup builds the seat our own system reserves for this job — WORD: a durable, addressed inbox. The bridge rule: a notification fires when a today-trigger fires on a record — the same fill = triggeredToday discipline that lights a Signal anywhere else. It inherits that Signal's colour = state, names its source as a Ref, and a tap teleports you to that source — never a Gate here, it's read-only until you land. The built per-record message thread (reach-the-person's dead code) lives on this surface too, since it renders nowhere today.

Judgment calls made for this first draft, flagged so Jac can push back:
  • Chips here are Signals, not Gates — read-only, click teleports to the source. You don't change status from the inbox, only at the record itself; that's a deliberate restriction, not an oversight.
  • Filled vs outline follows attention, not group — Needs-you-now and Waiting both fill (they're asking for something); Handled uses the calm outline-green already used for "reporting/insured" elsewhere.
  • Unread is three redundant cues (dot + bold + brighter card), never colour alone, per the contrast rule.
  • Type filter chips fall back to orange (not a status colour) since "type" isn't a state — matches the toggle rule's stated fallback exactly.
  • The flood state collapses "Handled" by default — a made-up call, not yet in the spec: read items are the one group allowed to stay hidden so volume doesn't bury what's still live.
  • "Durable" is asserted, not built — this mockup has no backend; the real version needs a persisted table (not state memory) so a reload or a different device doesn't lose the inbox.
  • Message thread is shown already expanded for one item so the reach-the-person tie-in is visible without a click; in the shipped version it would start collapsed like the others.
