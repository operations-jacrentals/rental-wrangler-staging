How it reads

  • The tile numbers never change. Compare the first two cards — identical figures ($1,285 / 3 / $700 / 14d); Member Mode only adds the arrows (and the banner for the $ figure). It never recomputes or drops a number. green ▼ = member savings (join pitch), red ▲ = retail penalty (cancel deterrent).
  • On by default for members (their baseline). Turning it off is the retention move — red arrows show how much leaving costs. For non-members it's off by default; turning it on is the pitch.
  • OPEN · ALL · TRANSACTIONS replaces the section title. Open = unpaid invoices · All = every invoice · Transactions = each payment across them.
